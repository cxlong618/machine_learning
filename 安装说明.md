# 🚀 机器学习环境安装指南

## 📋 快速安装

### 方法一：直接安装（推荐）
```bash
# 安装核心依赖
pip install numpy pandas matplotlib scikit-learn scipy seaborn

# 可选：安装Jupyter支持
pip install jupyter ipykernel
```

### 方法二：使用requirements文件
```bash
# 进入项目目录
cd F:\long\machine_learning\代码示例

# 安装所有依赖
pip install -r requirements.txt
```

## 🔧 详细安装步骤

### 1. 检查Python版本
```bash
python --version
# 建议使用Python 3.8+
```

### 2. 升级pip（重要）
```bash
python -m pip install --upgrade pip
```

### 3. 创建虚拟环境（推荐）
```bash
# 创建虚拟环境
python -m venv ml_env

# 激活虚拟环境
# Windows:
ml_env\Scripts\activate
# Mac/Linux:
source ml_env/bin/activate

# 在虚拟环境中安装依赖
pip install -r 代码示例/requirements.txt
```

### 4. 验证安装
```python
# 创建测试文件 test_install.py
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

print("✅ 所有库安装成功！")
print(f"NumPy版本: {np.__version__}")
print(f"Pandas版本: {pd.__version__}")

# 测试加载鸢尾花数据集
iris = load_iris()
print(f"鸢尾花数据集形状: {iris.data.shape}")
```

## 🛠️ 常见问题解决

### 问题1：pip安装缓慢
```bash
# 使用国内镜像源
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple/ numpy pandas matplotlib scikit-learn

# 或者永久配置
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple/
```

### 问题2：权限错误
```bash
# Windows: 使用 --user 参数
pip install --user numpy pandas matplotlib

# 或者以管理员身份运行CMD
```

### 问题3：matplotlib中文显示问题
```python
# 解决方法：下载中文字体
import matplotlib.pyplot as plt

# 设置中文字体
plt.rcParams['font.sans-serif'] = ['SimHei']  # Windows
# plt.rcParams['font.sans-serif'] = ['Arial Unicode MS']  # Mac
plt.rcParams['axes.unicode_minus'] = False
```

### 问题4：Jupyter无法启动
```bash
# 重新安装内核
python -m ipykernel install --user --name=ml_kernel

# 启动Jupyter
jupyter notebook
```

## 🚀 验证环境

### 运行测试代码
创建 `test_environment.py`:
```python
def test_all_libraries():
    print("🧪 测试机器学习环境...")

    # 测试NumPy
    import numpy as np
    arr = np.array([1, 2, 3, 4, 5])
    print(f"✅ NumPy测试通过: {arr.mean()}")

    # 测试Pandas
    import pandas as pd
    df = pd.DataFrame({'A': [1, 2], 'B': [3, 4]})
    print(f"✅ Pandas测试通过: {df.shape}")

    # 测试Matplotlib
    import matplotlib.pyplot as plt
    plt.figure(figsize=(2, 2))
    plt.plot([1, 2], [1, 2])
    plt.close()
    print("✅ Matplotlib测试通过")

    # 测试Scikit-learn
    from sklearn.datasets import load_iris
    iris = load_iris()
    print(f"✅ Scikit-learn测试通过: {iris.data.shape}")

    # 测试完整功能
    print("\n🎉 环境配置完成！可以开始学习机器学习了！")

if __name__ == "__main__":
    test_all_libraries()
```

运行测试：
```bash
python test_environment.py
```
